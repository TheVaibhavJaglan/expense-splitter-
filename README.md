# Smart Expense Splitter

## Problem
Managing shared expenses among students is difficult due to random payments and lack of tracking.

## Solution
This project helps users:
- Add shared expenses
- Track who paid
- Calculate balances automatically

## Features
- Simple CLI interface
- Offline usage
- JSON-based storage

## How to Run
1. Clone repo
2. Run:
   python main.py

## Example
Input:
A pays 300 for A, B, C

Output:
Balances updated automatically

## Learning Outcomes
- File handling in Python
- Modular coding
- Problem-solving
